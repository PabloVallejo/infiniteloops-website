About Infiniteloops

Infiniteloops is a technological service for developers to speed up the deployment and execution of process that will run periodically (every minute, every hour, every day or every month). We provide a high-availability compute infrastructure and performs all maintenance tasks for the platform. Developer only needs to focus on code not in infrastructure.

This idea borned in AngelHack Bogota, it was created by developers for developers. We had the need to deploy small service like scrappers that posted on social networks, ping bots to wake up a heroku server and other periodically tasks. We usually deployed this servers on Heroku or AWS but it was really complicated to do that, now we can do it in an easy way with just some commands and using our loved git we can have the insfrastructure needed.
